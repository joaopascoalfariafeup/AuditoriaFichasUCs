Pontos fortes

  • Tema relevante e atual (análise de dados em Python) adequado ao contexto académico
  • Abordagem e-learning bem estruturada com progressão modular e atividades diversificadas
  • Utilização de dados reais (Pordata, INE) que aproxima a aprendizagem de contextos práticos
  • Bibliografia adequada e acessível através da biblioteca da universidade
  • Carga horária total coerente com os 1,5 ECTS (40,5 horas)

Pontos fracos

  • Resultados de aprendizagem vagos, sem verbos de ação específicos da taxonomia de Bloom
  • Programa excessivamente genérico, sem detalhamento dos conteúdos específicos a abordar
  • Sistema de avaliação contraditório: menciona "avaliação distribuída com exame final" mas depois indica "100% exame"
  • Critério de frequência inadequado (85% nos testes de transição é uma classificação, não condição de frequência)
  • Ausência de mapeamento explícito entre objetivos, programa e avaliação
  • Incoerências entre versões PT e EN em alguns aspetos

Sugestões de melhoria

  • Reformular os resultados de aprendizagem usando verbos de ação específicos (ex: "aplicar", "analisar", "criar") e tornando-os mais mensuráveis
  • Detalhar o programa especificando os conteúdos de cada um dos cinco módulos mencionados
  • Clarificar completamente o sistema de avaliação, definindo se existe realmente avaliação distribuída ou se é apenas exame final
  • Corrigir as condições de frequência, substituindo critérios de classificação por condições adequadas (ex: participação nos quizzes obrigatórios)
  • Incluir mapeamento explícito entre resultados de aprendizagem, tópicos programáticos e componentes de avaliação
  • Harmonizar completamente as versões PT e EN, eliminando inconsistências
  • Especificar melhor as "sessões de apoio e feedback" nas metodologias de ensino

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:

  • Aplicar os conceitos fundamentais da linguagem Python para manipulação e análise de dados
  • Utilizar bibliotecas especializadas (pandas, numpy, matplotlib) para processamento e visualização de dados
  • Importar e exportar dados entre Python e folhas de cálculo (MS-Excel)
  • Analisar séries de dados reais provenientes de fontes como Pordata e INE
  • Resolver problemas práticos de gestão e análise de dados aplicando as ferramentas aprendidas

Programa

O programa está estruturado em cinco módulos sequenciais:

  • Módulo 1: Fundamentos de programação em Python (variáveis, estruturas de controlo, funções)
  • Módulo 2: Manipulação de dados com pandas (DataFrames, operações básicas, limpeza de dados)
  • Módulo 3: Análise numérica com numpy (arrays, operações matemáticas, estatísticas descritivas)
  • Módulo 4: Visualização de dados com matplotlib (gráficos básicos, personalização, interpretação)
  • Módulo 5: Interface com folhas de cálculo e análise de dados reais (importação/exportação Excel, casos práticos com dados Pordata/INE)

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

  • Quizzes de progressão (Q): Conjuntos de questões online no final de cada módulo (Peso: 40%)
  • Exame final (EF): Prova prática individual em sala de computadores, com duração de 1h30m (Peso: 60%; Nota mínima: 45%)

A classificação final é calculada como: CF = round(0.40 × Q + 0.60 × EF; 0)

Obtenção de frequência

Para acesso ao exame final, os estudantes têm de:

  • Realizar todos os quizzes de progressão dos cinco módulos
  • Obter classificação mínima de 50% na média dos quizzes de progressão
