Pontos fortes

  • Tema relevante e atual (análise de dados em Python) adequado ao contexto académico
  • Abordagem e-learning bem estruturada com diferentes tipos de atividades (vídeos, exercícios, quizzes, sessões de apoio)
  • Bibliografia adequada e acessível através da biblioteca da universidade
  • Carga horária realista para 1,5 ECTS (40,5 horas totais)
  • Referência específica a fontes de dados reais (Pordata e INE)

Pontos fracos

  • Resultados de aprendizagem vagos, sem verbos de ação específicos da taxonomia de Bloom
  • Programa excessivamente genérico, sem detalhamento dos conteúdos específicos
  • Contradição crítica entre a descrição de "avaliação distribuída" e a fórmula final de "100% exame"
  • Critério de frequência inadequado (85% nos testes) confunde acesso à avaliação com resultados
  • Ausência de informação sobre avaliação especial e épocas de recurso
  • Versões PT/EN com algumas inconsistências menores

Sugestões de melhoria

  • Reformular os resultados de aprendizagem usando verbos de ação específicos (ex: "aplicar", "analisar", "criar") e tornando-os mais mensuráveis
  • Detalhar o programa especificando os conteúdos de cada um dos cinco módulos mencionados
  • Clarificar o sistema de avaliação: se há quizzes obrigatórios, estes devem ter peso na classificação final ou servir apenas como pré-requisito
  • Corrigir o critério de frequência para focar no acesso à avaliação (ex: "completar todos os módulos") em vez de classificações mínimas
  • Especificar as condições de avaliação especial, recurso e melhoria de classificação
  • Harmonizar as versões PT/EN, especialmente na secção de avaliação

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:

  • Aplicar os conceitos fundamentais da linguagem Python para manipulação e análise de dados
  • Utilizar bibliotecas especializadas (pandas, numpy, matplotlib) para processamento e visualização de dados
  • Importar e exportar dados entre Python e folhas de cálculo (MS-Excel)
  • Analisar séries de dados reais provenientes de fontes como Pordata e INE
  • Resolver problemas práticos de gestão e análise de grandes volumes de dados

Programa

O programa está estruturado em cinco módulos sequenciais:

  • Módulo 1: Fundamentos de programação em Python (variáveis, estruturas de controlo, funções)
  • Módulo 2: Manipulação de dados com pandas (DataFrames, operações básicas, limpeza de dados)
  • Módulo 3: Análise numérica com numpy (arrays, operações matemáticas, estatísticas descritivas)
  • Módulo 4: Visualização de dados com matplotlib (gráficos básicos, personalização, exportação)
  • Módulo 5: Interface com folhas de cálculo e análise de dados reais (importação/exportação Excel, casos práticos com dados Pordata/INE)

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

  • Quizzes de progressão (Q): Conjuntos de questões online para transição entre módulos (sem peso na classificação final, mas obrigatórios para progressão)
  • Exame final (EF): Prova prática individual em sala de computadores com duração de 1h30m (Peso: 100%)

Classificação final (CF) = EF (arredondada à unidade)

Obtenção de frequência

Para acesso ao exame final, os estudantes devem:

  • Completar todos os cinco módulos da UC
  • Realizar todos os quizzes de progressão entre módulos
