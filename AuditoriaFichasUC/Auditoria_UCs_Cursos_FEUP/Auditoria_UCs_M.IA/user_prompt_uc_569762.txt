# Ficha a avaliar - versão em português

## Unidade curricular
Competências transversais para engenharia: LaTeX

## Curso responsável
Competências transversais para engenharia: LaTeX

## Créditos ECTS
1,5

## Ano
1

## Semestre
1

## Objetivos
Esta unidade de formação proporciona oportunidades para a aquisição e desenvolvimento de competências do Sistema de Preparação de Documentos LaTeX, uma ferramenta muito útil na produção de textos matemáticos e científicos, devido a sua alta qualidade tipográfica, que também pode ser utilizada para produção de documentos pessoais, artigos, livros, teses, apresentações, entre outros.

## Resultados de aprendizagem e competências
<strong>COMPETÊNCIAS</strong><br />1. Escrever um artigo científico para uma revista internacional, usando o estilo LATEX fornecido pelo editor.<br />2. Escrever uma tese ou um livro, sem ter de se preocupar com o estilo e a formatação.<br />3. Escrever uma carta ou um memorando com os estilos predefinidos do LATEX.<br />4. Criar uma base de dados bibliográfica usando BibTeX.<br />5. Criar um índice remissivo para um livro, usando makeindex.<br />6. Preparar uma apresentação usando o estilo beamer.<br />7. Incluir equações matemáticas em páginas Web usando a libraria mathjax.<br />8. Incluir figuras e fotografias num documento LATEX.<br />9. Modificar e adaptar um estilo LATEX já existente.<br />10. Criar um documento LATEX distribuído em vários ficheiros separados.<br /><br /><strong>RESULTADO DA APRENDIZAGEM</strong><br />- No final desta unidade de formação, o/a participante deverá saber utilizar o LaTeX com proficiência;<br />- O/a participante deverá ser capaz de escrever um documento em LaTeX, contendo os<br />elementos comummente usados em textos matemáticos e científicos.

## Pré-requisitos e co-requisitos
O curso é de iniciação ao LaTeX, pelo que não são necessários quaisquer pré-requisitos.

Blocos de texto</strong> <br />- Listas. <br />- Tabelas.<br />- Blocos preformatados. <br />- Texto com várias colunas.<br />- Notas de pé de página. <br />- Outros blocos especiais.<br /><strong>5. Gráficos</strong> <br />- Inclusão de ficheiros gráficos e fotografias. <br />- Pacotes para criação de ilustrações e gráficos matemáticos<br /><strong>6. # Ficha a avaliar - versão em inglês

## Unidade curricular
Transferable skills for engineering: LaTex

## Curso responsável
Transferable skills for engineering: LaTex

## Créditos ECTS
1,5

## Ano
1

## Semestre
1

## Objetivos
This course provides opportunities for acquisition and development of skills of document preparation system LaTeX, a very useful tool in the production of mathematical and scientific texts, due to its high quality printing, which can also be used to produce personal documents, articles, books, theses, presentations, among others.

## Resultados de aprendizagem e competências
<strong>SKILLS<br /></strong>1. Write a scientific article for an international journal, using the LATEX style provided by the editor.<br />2. Write a thesis or a book, without having to worry about style and formatting.<br />3. Write a letter or memo using the predefined LATEX styles.<br />4. Create a bibliographic database using BibTeX.<br />5. Create an index for a book, using makeindex.<br />6. Prepare a presentation using the beamer style.<br />7. Include math equations in web pages using the mathjax library.<br />8. Include figures and photographs in a LATEX document.<br />9. Modify and adapt an existing LATEX style.<br />10. Create a LATEX document distributed in several separate files.<br /><br /><br /><strong>LEARNING OUTCOMES</strong><br />- At the end of this course, the participant should be able to use LaTeX proficiently;<br />- The participant should be able to write a document in LaTeX, containing the elements commonly used in mathematical and scientific texts.

## Pré-requisitos e co-requisitos
The course is an introductory course to LaTeX, so there are no prerequisites.

## Programa
<p><strong>Programme</strong></p>
<p><strong>1. Introduction</strong><br />- Advantages and disadvantages of LaTeX, by comparison with other systems.<br />- Installation<br />- Preparation of source files<br />- Processing<br />- Correction of errors<br /><strong>2. Styles</strong><br />- Styles and LaTeX packages.<br />- Chapters, sections and appendices.<br />- The CTAN file.<br /><strong>3. Special symbols</strong><br />- Symbols and special characters.<br />- Mathematical formulas.<br /><strong>4. Text blocks</strong><br />- Lists.<br />- Tables.<br />- Preformatted blocks.<br />- Text with multiple columns.<br />- Footer notes.<br />- Other special blocks.<br /><strong>5. Graphics</strong><br />- Graphics and photos inclusion.<br />- Package for creating illustrations and mathematical charts<br /><strong>6. Bibliography</strong><br />- Bibliographic databases.<br />- Styles.<br />- Quotations.<br /><strong>7. Index</strong><br />- Table of Contents.<br />- List of Figures.<br />- List of tables.<br />- Index.<br /><strong>8. Articles/Papers</strong><br />- Preparation of scientific papers using the styles provided by the editors of the journals.<br /><strong>9. Theses</strong><br />- Specific styles for theses and adaptation to FEUP's style.<br /><strong>10. Presentations</strong><br />- Preparation of slides and Web pages.<br /><strong>11. Extensions</strong><br />- Creation of new commands.<br />- Modification and adaptation packages and styles.</p>
