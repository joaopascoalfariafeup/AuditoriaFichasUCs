Pontos fortes

  • Resultados de aprendizagem bem definidos com verbos de ação apropriados (utilizar, desenvolver, construir, implementar) adequados ao contexto prático da UC
  • Sistema de avaliação equilibrado com múltiplas componentes (testes, trabalho prático, avaliação docente, quizzes) que reflete adequadamente a natureza prática da UC
  • Nota mínima no 2º teste (7,5 valores) garante domínio das competências essenciais de programação
  • Programa abrangente cobrindo temas fundamentais: VBA, programação orientada a objetos, estruturas de dados e bibliotecas Python
  • Bibliografia adequada e atual para o contexto da UC
  • Carga horária total correta (162h = 6 ECTS × 27h) com distribuição equilibrada entre contacto (34,6%) e trabalho autónomo
  • Versões PT/EN coerentes e bem alinhadas

Pontos fracos

  • Objetivos gerais demasiado genéricos ("fornecer base sólida de conhecimentos") sem especificar competências concretas
  • Programa apresentado de forma condensada, sem estruturação clara em tópicos numerados ou mapeamento explícito para objetivos
  • Metodologias de ensino descritas de forma genérica, sem detalhar atividades específicas ou estratégias pedagógicas concretas
  • Falta de clareza sobre o regime de avaliação em época de recurso para as componentes de avaliação distribuída

Sugestões de melhoria

  • Reformular os objetivos para serem mais específicos, por exemplo: "Esta UC visa desenvolver competências avançadas de programação, incluindo programação orientada a objetos, estruturas de dados e desenvolvimento de aplicações integradas"
  • Estruturar o programa em tópicos numerados (T1, T2, etc.) e incluir mapeamento explícito para os resultados de aprendizagem, seguindo o modelo da ficha de referência
  • Detalhar as metodologias de ensino especificando atividades concretas, por exemplo: "exercícios práticos de programação", "desenvolvimento incremental do projeto", "sessões de debugging colaborativo"
  • Clarificar na secção "Melhoria de classificação" como as componentes de avaliação distribuída podem ser recuperadas em caso de insucesso ou ausência
  • Especificar na fórmula de cálculo que a classificação final é arredondada à unidade, seguindo as práticas padrão

Proposta de Nova Redação

Objetivos

Esta unidade curricular visa desenvolver competências avançadas de programação de computadores, incluindo programação orientada a objetos, estruturas de dados fundamentais e desenvolvimento de aplicações integradas. Os estudantes irão consolidar conhecimentos de programação através da linguagem Python e ferramentas de produtividade como VBA, preparando-os para resolver problemas computacionais complexos no contexto da engenharia industrial.

Programa

São abordados os seguintes tópicos programáticos:
T1. Ferramentas de produtividade pessoal: programação em VBA para Excel, desenvolvimento de aplicações integradas.
T2. Programação orientada a objetos: conceitos fundamentais, classes, propriedades, métodos e eventos em Python.
T3. Estruturas de dados: tipos primitivos, pilhas (stacks), recursividade, listas ligadas, filas (queues).
T4. Algoritmos: métodos de ordenação e pesquisa.
T5. Bibliotecas Python: utilização de numpy, pandas, tkinter, matplotlib para desenvolvimento de aplicações.

Métodos de ensino e atividades de aprendizagem

A UC combina aulas teóricas de apresentação de conceitos com aulas teórico-práticas de resolução de exercícios e desenvolvimento de projetos. As atividades incluem:
- Exercícios práticos de programação em Python e VBA
- Desenvolvimento incremental de um projeto prático ao longo do semestre
- Sessões de debugging e discussão de soluções
- Quizzes regulares para consolidação de conhecimentos
- Trabalho autónomo de consulta bibliográfica e resolução de exercícios complementares
