(a) Escala: 0 - Ausente ou totalmente inadequado; 1 - Parcialmente adequado; 2 - Largamente adequado; 3 - Totalmente adequado.

Pontos fortes

  • Programa bem estruturado em módulos sequenciais, cobrindo adequadamente as funcionalidades essenciais do Excel
  • Metodologia de ensino a distância bem adaptada ao contexto de competências transversais
  • Carga horária realista para uma UC de 1,5 ECTS (40,5 horas totais)
  • Inclusão de elementos interativos como fóruns e quizzes online
  • Bibliografia adequada com recursos oficiais e exercícios práticos

Pontos fracos

  • Resultados de aprendizagem mal formulados, não seguindo a taxonomia de Bloom nem usando verbos de ação específicos
  • Sistema de avaliação contraditório entre "avaliação distribuída com exame final" e "100% exame"
  • Critério de 85% para progressão entre módulos é excessivamente elevado
  • Falta de clareza sobre o peso real da componente distribuída vs exame final
  • Inconsistências entre versões PT e EN (filtros vs tabelas dinâmicas no programa)
  • Formatação HTML inadequada em várias secções

Sugestões de melhoria

  • Reformular os resultados de aprendizagem usando verbos de ação específicos: "Aplicar", "Criar", "Analisar", "Resolver"
  • Clarificar o sistema de avaliação definindo pesos específicos para cada componente (ex: 30% avaliação distribuída + 70% exame final)
  • Reduzir o critério de progressão de 85% para 70% ou 75%, mais adequado ao contexto
  • Harmonizar as versões PT e EN, especificando se são abordados "filtros" e/ou "tabelas dinâmicas"
  • Corrigir a formatação HTML, removendo tags desnecessárias e elementos de tradução automática
  • Especificar melhor as condições de obtenção de frequência e melhoria de classificação

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:

  • Aplicar as funcionalidades básicas e avançadas do MS Excel para organização e manipulação de dados
  • Criar fórmulas utilizando referências relativas e absolutas e funções apropriadas
  • Desenvolver gráficos e aplicar formatação condicional para visualização de dados
  • Construir tabelas dinâmicas para análise de grandes volumes de dados
  • Resolver problemas práticos de gestão utilizando séries de dados de fontes como Pordata e INE

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

  • Avaliação distribuída (30%): Quizzes online no Moodle sobre os diferentes módulos. Os estudantes progridem para o módulo seguinte com classificação ≥ 70% no teste de transição.
  • Exame final (70%): Prova prática presencial de 1 hora em sala de computadores, abrangendo todos os tópicos programáticos.

Classificação final = 0,30 × Avaliação distribuída + 0,70 × Exame final

Obtenção de frequência

Para acesso ao exame final, os estudantes devem:

  • Completar todos os módulos da avaliação distribuída
  • Obter classificação ≥ 70% em pelo menos 80% dos testes de transição
