Pontos fortes

  • Resultados de aprendizagem bem detalhados e numerados, facilitando referenciação e mapeamento
  • Programa abrangente que cobre todo o ciclo de vida de projetos de data warehouse
  • Metodologia de ensino adequada ao caráter prático da UC, com aulas teórico-práticas em laboratório
  • Avaliação equilibrada entre componente prática (projeto em grupo) e teórica (exame individual)
  • Bibliografia atual e relevante para a área de data warehousing
  • Carga de trabalho realista e bem distribuída (162h = 27h × 6 ECTS)
  • Nota mínima no exame (7,5/20) garante verificação individual das competências

Pontos fracos

  • Objetivos gerais demasiado sucintos, não especificando claramente as competências a desenvolver
  • Formatação do programa com listas aninhadas desnecessárias dificulta a leitura
  • Inconsistências terminológicas entre versões PT/EN (ex: "NotaProjeto" vs "Project")
  • Falta de mapeamento explícito entre tópicos programáticos e resultados de aprendizagem
  • Condições de frequência pouco claras (referência vaga a "avaliação distribuída")

Sugestões de melhoria

  • Expandir os objetivos gerais para especificar melhor as competências técnicas e metodológicas a desenvolver no contexto de sistemas de informação analíticos
  • Reformatar o programa numa lista simples, eliminando as listas aninhadas desnecessárias
  • Uniformizar a terminologia entre versões PT/EN (usar "Nota do Projeto" em vez de "NotaProjeto")
  • Adicionar mapeamento explícito entre tópicos programáticos e resultados de aprendizagem (ex: "→O1-O3")
  • Clarificar as condições de frequência, especificando que se refere à participação no projeto laboratorial
  • Corrigir erro tipográfico "tranformação" para "transformação" no programa

Proposta de Nova Redação

Objetivos

Esta unidade curricular tem como objetivo dotar os estudantes das competências técnicas e metodológicas necessárias para projetar, implementar e explorar sistemas de informação analíticos, com particular foco em armazéns de dados (data warehouses). Os estudantes desenvolverão competências práticas em modelação dimensional, processos ETL, tecnologias OLAP e visualização de dados, aplicando metodologias de gestão de projetos adequadas ao contexto de sistemas analíticos.

Programa

  • Indicadores de desempenho (→O1, O2)
  • Dimensões e hierarquias (→O3)
  • Modelação Conceptual do Armazém de Dados: o modelo MultiDim (→O3)
  • A Enterprise Data Warehouse: implementação numa base de dados relacional (→O3, O4)
  • Kimball Data Marts: implementação numa base de dados relacional (→O3, O4)
  • Extração, transformação e carregamento de dados: experiências com uma ferramenta ETL (→O4)
  • O modelo multi-dimensional (→O7)
  • Exemplos com um servidor OLAP (→O7, O8)
  • A linguagem MDX (→O8)
  • Como desenhar Dashboards: exemplos com uma ferramenta de desenho de dashboards (→O5)
  • Os princípios básicos do desenvolvimento ágil de projetos de armazenamento de dados (→O1)
  • Utilização de técnicas de análise de dados sobre cubos OLAP (→O8)
  • Requisitos de metadados (→O9)

Obtenção de frequência

Para acesso ao exame final, os estudantes têm de participar no projeto laboratorial (trabalho prático em grupo).
