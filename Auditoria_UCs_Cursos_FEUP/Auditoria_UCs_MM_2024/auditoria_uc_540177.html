Pontos Fortes

  • Metodologia pedagógica inovadora: A abordagem construcionista com gamificação é adequada para uma UC de programação, promovendo o envolvimento ativo dos estudantes
  • Programa bem estruturado: Os tópicos programáticos cobrem adequadamente os fundamentos da programação, desde conceitos básicos até aplicações multimédia
  • Avaliação equilibrada: O peso de 30% para o exame final e 70% para trabalho prático é apropriado para uma UC prática
  • Avaliação automática: O uso de correção automática nas provas práticas garante objetividade e feedback imediato
  • Coerência entre versões: As versões portuguesa e inglesa são consistentes

Pontos Fracos

  • Objetivos de aprendizagem inadequados: Não seguem a taxonomia de Bloom, são vagos e não são verificáveis. A estrutura conhecimentos/atitudes/aptidões não corresponde a resultados de aprendizagem
  • Falta de alinhamento pedagógico explícito: Não existe mapeamento claro entre objetivos, tópicos programáticos e métodos de avaliação
  • Critérios de avaliação pouco claros: Especialmente na "avaliação contínua" através de pontos de gamificação - falta transparência sobre como os pontos são atribuídos
  • Ausência de notas mínimas: Não são especificadas notas mínimas para as componentes de avaliação
  • Detalhamento insuficiente: Falta especificação sobre o número e natureza das atividades de aprendizagem

Sugestões de Melhorias

  • Reformular os objetivos de aprendizagem: Usar verbos de ação da taxonomia de Bloom (ex: "Implementar algoritmos básicos", "Analisar problemas computacionais", "Criar aplicações interativas simples")
  • Criar mapeamento explícito: Estabelecer correspondência clara entre objetivos, tópicos programáticos e componentes de avaliação
  • Detalhar critérios de avaliação: Especificar como são atribuídos os pontos na gamificação e definir rubricas claras
  • Definir notas mínimas: Estabelecer nota mínima para as provas práticas (ex: 45%) para garantir competências essenciais
  • Especificar atividades: Detalhar o tipo e número de exercícios, projetos e atividades gamificadas
  • Melhorar a descrição das metodologias: Explicitar como as atividades contribuem para cada objetivo de aprendizagem
