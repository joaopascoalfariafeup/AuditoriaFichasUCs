Metodologias de ensino e aprendizagem As metodologias são adequadas ao tipo de UC (tutoria individual), mas a descrição é muito superficial e repetitiva. Não especifica como as sessões de tutoria contribuem para os diferentes objetivos de aprendizagem. 1
Metodologias de avaliação A avaliação está estruturada em fases progressivas (plano, projeto, versão completa), o que é apropriado. Contudo, falta clareza na descrição dos critérios, pesos inconsistentes (10%+40%≠50%) e ausência de mapeamento com os objetivos. 1
Apreciação global A ficha apresenta uma estrutura básica adequada ao tipo de UC, mas carece de detalhamento, clareza e alinhamento pedagógico. Necessita de revisão significativa para melhorar a qualidade pedagógica. 1.25

Síntese da Avaliação

Pontos Fortes

  • Os objetivos de aprendizagem são relevantes e específicos para uma UC de investigação
  • A estrutura de avaliação em fases progressivas (plano → projeto → versão completa) é pedagogicamente adequada
  • A metodologia de tutoria individual é apropriada para o tipo de trabalho desenvolvido
  • A carga horária (18 ECTS) é realista para o tipo de trabalho exigido

Pontos Fracos

  • Inconsistência nos pesos de avaliação (10% + 40% ≠ 50% declarado para trabalho escrito)
  • Programa muito genérico e pouco estruturado
  • Ausência de mapeamento explícito entre objetivos, conteúdos e avaliação
  • Metodologias de ensino descritas de forma muito superficial
  • Falta de critérios de avaliação específicos
  • Inconsistências entre as versões portuguesa e inglesa
  • Objetivos não formulados com verbos de ação da taxonomia de Bloom

Sugestões de Melhoria

  • Corrigir os pesos de avaliação: Esclarecer se são 50% trabalho escrito + 50% trabalho prático, ou 10% + 40% + 50% (totalizando 100%)
  • Reformular os objetivos: Utilizar verbos de ação específicos (ex: "analisar", "sintetizar", "avaliar") alinhados com os níveis cognitivos pretendidos
  • Detalhar o programa: Especificar os conteúdos abordados nas sessões de tutoria e criar mapeamento explícito com os objetivos
  • Expandir as metodologias de ensino: Descrever como as diferentes atividades contribuem para cada objetivo de aprendizagem
  • Definir critérios de avaliação: Especificar os critérios para cada componente de avaliação e como avaliam os diferentes objetivos
  • Harmonizar as versões linguísticas: Garantir coerência total entre as versões portuguesa e inglesa
  • Incluir bibliografia: Adicionar referências bibliográficas de apoio à metodologia de investigação
