Síntese da Avaliação

Pontos Fortes

  • Objetivos bem definidos: Os resultados de aprendizagem são claros, específicos e usam verbos de ação apropriados, cobrindo desde competências básicas até avançadas em programação.
  • Progressão lógica do conteúdo: Os tópicos programáticos seguem uma sequência pedagógica adequada, desde conceitos fundamentais até temas mais complexos.
  • Metodologia apropriada: A combinação de aulas teóricas e práticas é adequada para uma disciplina de programação, especialmente para iniciantes.
  • Contextualização: A unidade curricular está bem contextualizada no âmbito da Engenharia Aeroespacial.
  • Coerência entre versões: As versões em português e inglês são consistentes.

Pontos Fracos

  • Sistema de avaliação confuso: A fórmula de cálculo usa nomenclaturas pouco claras (DP, μT, MT) sem explicação adequada do que cada componente avalia especificamente.
  • Ausência de mapeamento explícito: Não existe mapeamento claro entre tópicos programáticos, metodologias e objetivos de aprendizagem.
  • Falta de diversidade na avaliação: O sistema parece focado principalmente em testes, sem componentes de projeto ou trabalho prático claramente identificadas.
  • Critérios de avaliação vagos: Não são especificados critérios claros para cada componente de avaliação.
  • Ausência de notas mínimas: Não são definidas notas mínimas para as diferentes componentes.

Sugestões de Melhorias

  • Clarificar o sistema de avaliação: Substituir as siglas (DP, μT, MT) por descrições claras e explicar o que cada componente avalia especificamente.
  • Incluir mapeamento explícito: Criar uma tabela que relacione cada tópico programático e componente de avaliação com os objetivos de aprendizagem correspondentes.
  • Diversificar métodos de avaliação: Incluir componentes de projeto prático que permitam avaliar competências de nível cognitivo mais elevado.
  • Definir critérios de avaliação: Especificar critérios claros para cada componente, incluindo notas mínimas quando aplicável.
  • Melhorar a descrição das metodologias: Detalhar melhor como as atividades de aprendizagem contribuem para os objetivos específicos.
  • Incluir avaliação supervisionada individual: Garantir que existe pelo menos uma componente de avaliação individual supervisionada claramente identificada.
