Pontos Fortes

  • Programa bem estruturado: Os tópicos programáticos estão organizados de forma lógica, cobrindo adequadamente transferência de calor e massa com progressão clara dos conceitos básicos para aplicações práticas.
  • Aplicações práticas relevantes: Excelente integração de exemplos aplicados à bioengenharia (queimaduras na pele, pasteurização, hemodiálise, etc.).
  • Diversidade de métodos de avaliação: Combina testes, trabalho prático e componente laboratorial, oferecendo múltiplas oportunidades de demonstração de competências.
  • Flexibilidade no sistema de avaliação: Permite diferentes percursos de avaliação conforme o desempenho no primeiro teste.
  • Componente laboratorial obrigatória: Garante experiência prática essencial na área.

Pontos Fracos

  • Objetivos de aprendizagem inadequados: Não seguem a taxonomia de Bloom, são vagos e não mensuráveis. Faltam verbos de ação específicos.
  • Ausência de mapeamento: Não existe alinhamento explícito entre objetivos, tópicos programáticos, metodologias e avaliação.
  • Peso excessivo do exame: O exame representa 65% da classificação, ultrapassando o limite recomendado de 75% para o conjunto das componentes supervisionadas.
  • Condições de frequência vagas: Apenas menciona "frequência das aulas" sem especificar percentagem mínima de assiduidade.
  • Descrição genérica das metodologias: As metodologias de ensino são descritas de forma superficial, sem detalhar como contribuem para os objetivos específicos.

Sugestões de Melhorias

  • Reformular objetivos de aprendizagem: Redigir objetivos claros, específicos e mensuráveis usando verbos da taxonomia de Bloom (ex: "Calcular", "Analisar", "Avaliar", "Aplicar").
  • Criar mapeamento explícito: Estabelecer correspondência clara entre objetivos de aprendizagem, tópicos programáticos e componentes de avaliação.
  • Reequilibrar pesos de avaliação: Reduzir o peso do exame e aumentar o peso das componentes de avaliação distribuída.
  • Especificar condições de frequência: Definir claramente a percentagem mínima de assiduidade (ex: 75% das aulas).
  • Detalhar metodologias de ensino: Explicar como cada metodologia contribui para objetivos específicos de aprendizagem.
  • Simplificar sistema de avaliação: Considerar simplificar as múltiplas equações e percursos de avaliação para maior clareza.
  • Incluir nota mínima para trabalho laboratorial: Especificar se existe nota mínima para a componente laboratorial.
