Síntese da Avaliação

Pontos Fortes

  • Objetivos de aprendizagem bem definidos e específicos, cobrindo competências técnicas relevantes
  • Programa estruturado de forma lógica e progressiva, desde conceitos teóricos até implementação prática
  • Boa coerência entre as versões portuguesa e inglesa
  • Projeto prático que integra os conhecimentos da unidade curricular
  • Carga horária realista e bem distribuída

Pontos Fracos

  • Ausência de mapeamento explícito entre objetivos de aprendizagem e tópicos programáticos
  • Metodologias de ensino descritas de forma muito genérica
  • Sistema de avaliação demasiado simples, com apenas duas componentes
  • Falta de mapeamento entre componentes de avaliação e objetivos de aprendizagem
  • Ausência de critérios de avaliação detalhados
  • Não há componentes de avaliação formativa ou participação
  • Condições de frequência pouco claras quanto à assiduidade

Sugestões de Melhorias

  • Incluir mapeamento explícito entre cada objetivo de aprendizagem e os tópicos programáticos correspondentes
  • Detalhar as metodologias de ensino, especificando que atividades contribuem para que objetivos
  • Diversificar as componentes de avaliação, incluindo elementos intermédios (ex: testes, apresentações)
  • Mapear cada componente de avaliação para os objetivos de aprendizagem específicos
  • Definir critérios de avaliação claros para o projeto e suas componentes
  • Incluir uma componente de participação/assiduidade
  • Clarificar as condições de assiduidade para obtenção de frequência
  • Considerar dividir o projeto em mais entregas intermédias para melhor acompanhamento
  • Especificar se existe possibilidade de avaliação de recurso para o projeto
