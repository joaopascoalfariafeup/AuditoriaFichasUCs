Pontos fortes

  • Objetivos do curso claramente definidos e bem estruturados, abrangendo tanto competências técnicas como transversais
  • Excelente cobertura das áreas mencionadas nos objetivos: Gestão de Operações, Informática/Indústria 4.0, Métodos Quantitativos e Gestão de Empresas
  • Estrutura flexível com grupos de opção que permite especialização em diferentes áreas
  • Progressão lógica do 1º para o 2º ano, culminando na dissertação que integra conhecimentos adquiridos
  • Inclusão de UCs que desenvolvem competências de liderança e comunicação (Comportamento Organizacional, Seminários)
  • Forte componente prática através de projetos e casos de estudo empresariais
  • Integração de temas contemporâneos como sustentabilidade, economia circular e Objetivos de Desenvolvimento Sustentável

Pontos fracos

  • Algumas sobreposições entre UCs de análise de dados (Analítica Empresarial, Ciência de Dados para Empresas, Engenharia de Grandes Dados)
  • Possível redundância entre UCs de estatística (Estatística Multivariada e componentes estatísticas de outras UCs)
  • Falta de informação sobre UCs obrigatórias do 1º ano, dificultando a avaliação completa da sequência

Sugestões de melhoria

  • Clarificar a diferenciação entre as UCs de análise de dados, definindo focos específicos para cada uma (ex: Analítica Empresarial para métodos tradicionais, Ciência de Dados para técnicas avançadas, Engenharia de Grandes Dados para infraestruturas)
  • Considerar a criação de pré-requisitos mais específicos entre UCs relacionadas para garantir progressão adequada
  • Reforçar a articulação entre UCs teóricas e práticas através de projetos integrados
  • Desenvolver mecanismos de coordenação entre docentes das UCs com conteúdos relacionados para evitar redundâncias desnecessárias
  • Considerar a inclusão de mais UCs focadas em tecnologias emergentes (IA, IoT, blockchain) para reforçar a componente Indústria 4.0
