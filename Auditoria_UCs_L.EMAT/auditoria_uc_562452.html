Pontos fortes

  • Abordagem e-learning adequada para uma UC de 1,5 ECTS com foco prático em Python
  • Utilização de dados reais (Pordata, INE) confere relevância prática
  • Bibliografia adequada e acessível através da biblioteca
  • Carga horária coerente com os créditos ECTS (40,5h para 1,5 ECTS)
  • Sessões de apoio opcionais demonstram preocupação com o suporte aos estudantes

Pontos fracos

  • Objetivos e resultados de aprendizagem vagos, sem verbos de ação específicos nem níveis cognitivos definidos
  • Programa excessivamente genérico, sem especificação de conteúdos concretos
  • Contradição fundamental: declara "avaliação distribuída com exame final" mas a fórmula é "100% exame"
  • Critério de 85% para progressão entre módulos é excessivamente elevado e inadequado para obtenção de frequência
  • Ausência de mapeamento entre objetivos, programa e avaliação
  • Inconsistências entre versões PT/EN (formatação HTML diferente)

Sugestões de melhoria

  • Reformular os resultados de aprendizagem com verbos de ação específicos (ex: "Implementar", "Analisar", "Visualizar") e especificar níveis de proficiência
  • Detalhar o programa especificando tópicos concretos de Python (estruturas de dados, manipulação de DataFrames, visualização, etc.)
  • Clarificar o sistema de avaliação: se há quizzes distribuídos, devem ter peso na classificação final ou servir apenas para progressão
  • Reduzir o critério de progressão de 85% para 50-60% ou reformular como condição de frequência
  • Estabelecer mapeamento explícito entre objetivos, tópicos programáticos e componentes de avaliação
  • Uniformizar a formatação HTML entre versões PT/EN
  • Especificar o tipo de questões do exame final (práticas em computador, teóricas, mistas)

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:
R1. Implementar soluções básicas em Python utilizando estruturas de dados fundamentais (listas, dicionários, DataFrames).
R2. Utilizar bibliotecas especializadas (pandas, numpy, matplotlib) para manipulação e análise de dados.
R3. Importar, limpar e transformar conjuntos de dados de fontes externas (Excel, CSV, APIs).
R4. Criar visualizações básicas de dados utilizando matplotlib.
R5. Analisar séries temporais e dados estatísticos utilizando dados reais do INE e Pordata.
R6. Integrar Python com ferramentas de produtividade (MS-Excel) para automatização de tarefas.

Programa

O programa está estruturado em cinco módulos sequenciais:
Módulo 1: Fundamentos de Python - sintaxe básica, estruturas de controlo, funções (→R1)
Módulo 2: Estruturas de dados - listas, dicionários, introdução ao pandas e DataFrames (→R1, R2)
Módulo 3: Manipulação de dados - importação de ficheiros CSV/Excel, limpeza e transformação de dados (→R2, R3)
Módulo 4: Visualização de dados - criação de gráficos com matplotlib, análise exploratória (→R4, R5)
Módulo 5: Integração e automatização - interface Python-Excel, análise de dados do INE/Pordata (→R3, R5, R6)

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

1) Avaliação distribuída (AD) (Peso 40%):
- Quizzes de progressão: 5 quizzes online, um por módulo (Peso: 40%; →R1-R6)

2) Exame final (EF) (Peso 60%):
- Exame prático individual em sala de computadores, com duração de 1h30m (Peso: 60%; Nota mínima: 8,0 valores; →R1-R6)

A classificação final (CF) é calculada como:
CF = round(0.40 × AD + 0.60 × EF; 0)

Obtenção de frequência

Para acesso ao exame final, os estudantes devem:
- Realizar todos os 5 quizzes de progressão da avaliação distribuída
- Obter classificação mínima de 50% em cada quiz para progressão ao módulo seguinte
