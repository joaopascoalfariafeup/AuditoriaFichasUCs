Objetivos/resultados de aprendizagem Objetivos vagos e genéricos, sem verbos de ação específicos. Resultados de aprendizagem apresentados como lista de tópicos em vez de competências verificáveis. 1

(a) Escala: 0 - Ausente ou totalmente inadequado; 1 - Parcialmente adequado; 2 - Largamente adequado; 3 - Totalmente adequado.

Pontos fortes

  • Tópicos programáticos bem estruturados e adequados para uma formação em Excel, cobrindo desde conceitos básicos até funcionalidades avançadas como tabelas dinâmicas
  • Metodologia de ensino a distância bem descrita e apropriada para uma UC de competências transversais
  • Carga horária (40,5h) realista para 1,5 ECTS, respeitando aproximadamente as 27h por crédito
  • Foco em aplicações práticas com dados reais (Pordata, INE)

Pontos fracos

  • Objetivos e resultados de aprendizagem formulados de forma vaga, sem verbos de ação específicos nem critérios de verificação
  • Sistema de avaliação contraditório: declara "avaliação distribuída com exame final" mas depois indica "100% exame"
  • Critério de 85% para progressão entre módulos é excessivamente elevado e pode criar barreiras desnecessárias
  • Ausência de mapeamento explícito entre tópicos programáticos e objetivos de aprendizagem
  • Inconsistências entre versões PT/EN (versão inglesa menciona "Filters" que não aparece na portuguesa)
  • Bibliografia limitada e pouco diversificada

Sugestões de melhoria

  • Reformular os objetivos usando verbos de ação específicos (ex: "aplicar", "criar", "analisar") e tornar os resultados de aprendizagem verificáveis
  • Clarificar o sistema de avaliação: se é 100% exame, remover referência à "avaliação distribuída"; se há componentes distribuídas, especificar pesos e critérios
  • Reduzir o critério de progressão de 85% para um valor mais razoável (ex: 60-70%) ou justificar pedagogicamente este valor elevado
  • Adicionar mapeamento explícito entre tópicos programáticos e objetivos de aprendizagem
  • Harmonizar as versões PT/EN, decidindo se "Filtros/Filters" faz parte do programa
  • Expandir a bibliografia com recursos mais diversificados e atuais sobre análise de dados
  • Especificar melhor as condições de obtenção de frequência e os procedimentos de melhoria de classificação

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:
• Aplicar as funcionalidades básicas e avançadas do MS Excel para organização e manipulação de dados
• Criar fórmulas e funções para cálculos automatizados em folhas de cálculo
• Construir gráficos e aplicar formatação condicional para visualização de dados
• Desenvolver tabelas dinâmicas para análise de grandes volumes de dados
• Analisar séries de dados reais provenientes de fontes como Pordata e INE
• Resolver problemas práticos de gestão utilizando ferramentas de análise de dados

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

1) Avaliação distribuída (70%):
• Quizzes online sobre cada módulo do programa (peso: 30%)
• Projeto prático de análise de dados (peso: 40%)

2) Exame final presencial (30%):
• Prova prática em sala de computadores (duração: 1h)

Classificação final = 0,70 × Avaliação Distribuída + 0,30 × Exame Final

Nota: Para progressão entre módulos é necessária classificação mínima de 60% no quiz respetivo.

Obtenção de frequência

Para acesso ao exame final, os estudantes devem:
• Completar todos os módulos do programa com classificação mínima de 60% nos quizzes
• Entregar o projeto prático de análise de dados
