Proposta de Nova Redação

Objetivos

Esta unidade curricular tem como objetivo aprofundar conhecimentos avançados em ciência de dados, explorando técnicas e metodologias de ponta para análise de dados complexos e desenvolvimento de soluções inteligentes. Pretende-se capacitar os estudantes para enfrentar desafios reais em projetos de ciência de dados de grande escala e complexidade.

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de: O1. Aplicar técnicas avançadas de machine learning e deep learning para resolver problemas complexos de ciência de dados. O2. Implementar e otimizar algoritmos de processamento de big data usando frameworks distribuídos. O3. Desenvolver soluções de processamento de linguagem natural para análise de texto não estruturado. O4. Projetar e implementar sistemas de recomendação e análise preditiva. O5. Avaliar criticamente a qualidade e limitações de modelos de dados complexos. O6. Comunicar eficazmente resultados de análises avançadas a audiências técnicas e não técnicas.

Programa

T1. Técnicas avançadas de machine learning: ensemble methods, transfer learning, meta-learning (→O1). T2. Deep learning aplicado: redes neurais convolucionais, recorrentes e transformers (→O1). T3. Big data analytics: processamento distribuído com Spark, Hadoop, técnicas de streaming (→O2). T4. Processamento de linguagem natural: análise semântica, modelos de linguagem, sentiment analysis (→O3). T5. Sistemas de recomendação: collaborative filtering, content-based, hybrid approaches (→O4). T6. Análise preditiva avançada: time series forecasting, anomaly detection (→O4). T7. Validação e interpretabilidade de modelos: técnicas de explicabilidade, fairness, robustez (→O5). T8. Visualização avançada e storytelling com dados (→O6).

Métodos de ensino e atividades de aprendizagem

As aulas combinam exposição teórica dos conceitos avançados com sessões práticas intensivas de implementação. Os estudantes desenvolvem projetos aplicados usando datasets reais e ferramentas profissionais. Inclui seminários com especialistas da indústria e apresentações de projetos pelos estudantes, promovendo a discussão crítica e partilha de experiências.

Tipo de avaliação

Avaliação distribuída sem exame final

Fórmula de cálculo da classificação final

A avaliação compreende: 1) Projeto individual (PI): Implementação de solução avançada de ciência de dados (Peso: 40%; Nota mínima: 45%; →O1-O5) 2) Projeto de grupo (PG): Desenvolvimento de sistema completo de análise de dados (Peso: 40%; Nota mínima: 45%; →O1-O6) 3) Apresentações e discussão (AD): Apresentação oral dos projetos e participação em discussões (Peso: 20%; →O6) CF = round(0.40 * PI + 0.40 * PG + 0.20 * AD; 0)
