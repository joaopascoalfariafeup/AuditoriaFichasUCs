Pontos fracos

  • Objetivos redigidos de forma vaga, sem verbos de ação específicos nem clareza sobre o que se pretende alcançar

  • Organizar os tópicos programáticos de forma hierárquica, com subtópicos detalhados e mapeamento explícito para os objetivos (ex: T1→O1, T2→O2)
  • Detalhar as metodologias de ensino, especificando como cada método contribui para desenvolver as competências pretendidas

Proposta de Nova Redação

Objetivos

Esta unidade curricular tem como objetivo principal capacitar os estudantes para compreender e aplicar técnicas avançadas de Inteligência Artificial baseadas em conhecimento do domínio. Especificamente, pretende-se que os estudantes sejam capazes de desenvolver sistemas baseados em conhecimento, aplicar técnicas de enriquecimento de dados usando conhecimento semântico, e implementar algoritmos de aprendizagem computacional multi-relacionais para resolver problemas complexos em domínios científicos e industriais.

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:
O1. Analisar e implementar sistemas baseados em conhecimento, incluindo sistemas periciais.
O2. Construir e manipular redes semânticas e knowledge graphs para representação de conhecimento.
O3. Avaliar a interpretabilidade e explicabilidade de modelos de aprendizagem computacional usando métricas apropriadas.
O4. Aplicar técnicas de enriquecimento de conjuntos de dados usando conhecimento do domínio.
O5. Implementar técnicas de elicitação de conhecimento para modelos opacos (black-box).
O6. Desenvolver soluções de aprendizagem computacional para dados heterogéneos.
O7. Aplicar algoritmos de aprendizagem computacional em ambientes distribuídos.
O8. Implementar e avaliar algoritmos multi-relacionais para problemas complexos.

Programa

São abordados os seguintes tópicos programáticos, organizados e mapeados para os objetivos de aprendizagem:
T1. Sistemas Baseados em Conhecimento: arquiteturas, motores de inferência e sistemas periciais (→O1)
T2. Representação de Conhecimento: redes semânticas, ontologias e knowledge graphs (→O2)
T3. Interpretabilidade em IA: métricas de explicabilidade, técnicas de interpretação de modelos (→O3)
T4. Enriquecimento de Dados: técnicas de integração de conhecimento do domínio em conjuntos de dados (→O4)
T5. Elicitação de Conhecimento: métodos para extrair conhecimento de modelos opacos (→O5)
T6. Aprendizagem com Dados Heterogéneos: metodologias para análise de dados multi-modais (→O6)
T7. Aprendizagem Distribuída: algoritmos e arquiteturas para ambientes distribuídos (→O7)
T8. Aprendizagem Multi-Relacional: algoritmos ILP, Graph Neural Networks (GNN) (→O8)
T9. Aplicações: problemas científicos em ciências da vida, medicina, análise de texto e reconhecimento de imagens (→O1-O8)

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

1) Projeto Individual (P) (Peso: 65%): Desenvolvimento de um projeto aplicando os conhecimentos adquiridos a um problema real. Inclui relatório escrito e implementação prática. (→O1-O8)

2) Apresentação e Discussão do Projeto (A) (Peso: 30%): Apresentação oral obrigatória do projeto com discussão individual para verificar a compreensão dos conceitos aplicados. (→O1-O8)

3) Participação nas Aulas (PA) (Peso: 5%): Participação ativa nas discussões e atividades práticas. (→O1-O8)

Classificação Final (CF) = 0.65 × P + 0.30 × A + 0.05 × PA
