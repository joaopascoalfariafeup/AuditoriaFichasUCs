Pontos fortes

  • Programa abrangente e bem estruturado, cobrindo métodos computacionais fundamentais (integrais, derivadas, equações diferenciais, Monte Carlo, etc.)
  • Bibliografia adequada com referências clássicas da área (Newman, Gould, Chapra)
  • Software apropriado para a UC (Python, numpy, matplotlib, scipy)
  • Carga horária coerente com os 6 ECTS (162h total, 35% de contacto)
  • Coerência entre versões portuguesa e inglesa

Pontos fracos

  • Objetivos extremamente genéricos ("introduzir métodos computacionais") sem especificação clara do que os estudantes devem alcançar
  • Resultados de aprendizagem redigidos como um parágrafo corrido, sem numeração nem verbos de ação adequados
  • Fórmula de avaliação matematicamente complexa e pouco transparente para estudantes
  • Ausência de especificação das componentes concretas da avaliação contínua
  • Metodologias de ensino descritas de forma muito superficial
  • Problemas técnicos de formatação (tags HTML indesejadas)

Sugestões de melhoria

  • Reformular os objetivos especificando competências concretas: "Ao completar esta UC, os estudantes devem ser capazes de aplicar métodos numéricos para resolver problemas de física, implementar algoritmos em Python, e analisar criticamente resultados computacionais"
  • Reestruturar os resultados de aprendizagem em lista numerada com verbos de ação específicos (aplicar, implementar, analisar, comparar, validar)
  • Simplificar a fórmula de avaliação ou explicá-la com exemplos práticos, especificando claramente as componentes da avaliação contínua (trabalhos práticos, projetos, testes, etc.)
  • Detalhar as metodologias de ensino: "Aulas teóricas para apresentação dos fundamentos matemáticos, aulas práticas laboratoriais para implementação de algoritmos, projetos individuais de aplicação a problemas físicos específicos"
  • Estabelecer mapeamento explícito entre tópicos programáticos e resultados de aprendizagem
  • Corrigir problemas de formatação removendo tags HTML indesejadas

Proposta de Nova Redação

Objetivos

Esta unidade curricular tem como objetivo capacitar os estudantes para a aplicação de métodos computacionais na resolução de problemas de Física e Engenharia. Ao completar a UC, os estudantes devem dominar técnicas numéricas fundamentais, saber implementá-las em linguagens de programação, e ser capazes de analisar criticamente os resultados obtidos.

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:

O1. Identificar e formular problemas computacionais a partir de equações físicas.
O2. Selecionar algoritmos numéricos apropriados para diferentes tipos de problemas (integração, diferenciação, sistemas de equações, equações diferenciais).
O3. Implementar algoritmos numéricos em Python utilizando bibliotecas científicas (numpy, scipy, matplotlib).
O4. Aplicar métodos de Monte Carlo para resolver problemas estocásticos em física.
O5. Analisar criticamente resultados computacionais, comparando com soluções analíticas e casos limite conhecidos.
O6. Validar implementações através de testes de convergência e análise de erros numéricos.

Métodos de ensino e atividades de aprendizagem

A UC combina aulas teóricas para apresentação dos fundamentos matemáticos dos métodos numéricos com aulas práticas laboratoriais para implementação hands-on dos algoritmos. Os estudantes desenvolvem projetos individuais aplicando os métodos a problemas físicos específicos, com acompanhamento tutorial para discussão de resultados e validação de implementações.

Fórmula de cálculo da classificação final

Componentes de avaliação:

• Avaliação contínua (C) - 30%: Trabalhos práticos laboratoriais e projetos de implementação
• Exame final (E) - 70%: Prova escrita individual cobrindo fundamentos teóricos e resolução de problemas

Classificação final: F = 0.3 × C + 0.7 × E

Nota mínima no exame final: 8.0 valores
