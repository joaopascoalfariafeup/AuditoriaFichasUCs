Proposta de Nova Redação

Objetivos

Esta unidade curricular tem como objetivos: (1) consolidar e aprofundar as competências de programação orientada a objetos dos estudantes, com foco na linguagem Java; (2) capacitar os estudantes para o desenho e implementação das principais estruturas de dados lineares e não-lineares e respetivos algoritmos; (3) desenvolver competências de análise de complexidade algorítmica e seleção de estruturas de dados apropriadas para diferentes problemas; (4) aplicar técnicas algorítmicas fundamentais como recursividade, backtracking e dividir-para-conquistar na resolução de problemas práticos.

Resultados de aprendizagem e competências

Ao concluirem esta unidade curricular os estudantes deverão saber:

  • usar os princípios básicos de uma linguagem de programação orientada aos objetos, nomeadamente herança, polimorfismo, interfaces, classes do tipo coleção e iteradores.
  • escrever classes que implementem algumas estruturas de dados lineares, como sejam listas, pilhas, filas, conjuntos, dicionários, tabelas de hash e árvores binárias.
  • usar uma API que implemente listas, pilhas, filas, conjuntos, e tabelas de hash.
  • desenvolver algoritmos básicos associados às estruturas de dados estudadas.
  • usar algumas das técnicas algorítmicas, nomeadamente recursividade, pesquisa com retrocesso e dividir-para-conquistar.
  • fazer a análise de complexidade de algoritmos e identificar as principais classes de complexidade (competência básica).
  • aplicar os conhecimentos na resolução prática de problemas concretos.

Obtenção de frequência

Para obter frequência, os estudantes devem cumprir cumulativamente as seguintes condições:

  • responder a pelo menos 50% dos questionários online (quizzes);
  • ter assiduidade mínima de 50% às aulas dadas.
