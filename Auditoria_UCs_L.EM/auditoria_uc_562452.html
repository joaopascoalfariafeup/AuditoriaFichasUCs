Pontos fortes

  • Abordagem e-learning adequada para uma UC de 1,5 ECTS com foco prático em Python
  • Utilização de dados reais (Pordata e INE) confere relevância prática à aprendizagem
  • Bibliografia adequada e acessível através da biblioteca da universidade
  • Carga horária (40,5h) coerente com os créditos ECTS atribuídos
  • Coerência entre versões portuguesa e inglesa

Pontos fracos

  • Objetivos e resultados de aprendizagem formulados de forma vaga, sem verbos de ação específicos
  • Programa extremamente superficial, com apenas três tópicos genéricos
  • Contradição fundamental no sistema de avaliação: menciona "avaliação distribuída" mas a classificação final é 100% exame
  • Critério de 85% para progressão entre módulos é excessivamente elevado e pode ser desmotivador
  • Ausência de mapeamento entre objetivos, programa e avaliação
  • Falta de clareza sobre o papel real dos quizzes na avaliação final

Sugestões de melhoria

  • Reformular os resultados de aprendizagem usando verbos de ação específicos (ex: "Implementar", "Analisar", "Visualizar") e especificar níveis de proficiência
  • Detalhar o programa com tópicos específicos como: "Estruturas de dados em Python", "Manipulação de DataFrames com pandas", "Visualização de dados com matplotlib", etc.
  • Clarificar o sistema de avaliação: se a classificação final é 100% exame, especificar que os quizzes são apenas para progressão/frequência
  • Reduzir o critério de progressão de 85% para 70% ou criar critérios diferenciados por módulo
  • Incluir mapeamento explícito entre objetivos de aprendizagem e tópicos programáticos
  • Especificar melhor as "sessões de apoio e feedback" mencionadas nas metodologias de ensino

Proposta de Nova Redação

Resultados de aprendizagem e competências

No final da unidade curricular, os estudantes devem ser capazes de:
O1. Implementar algoritmos básicos em Python utilizando estruturas de dados fundamentais (listas, dicionários, conjuntos).
O2. Manipular e transformar dados utilizando a biblioteca pandas (importação, limpeza, filtragem, agregação).
O3. Criar visualizações de dados utilizando matplotlib e outras bibliotecas de visualização.
O4. Integrar Python com folhas de cálculo Excel para importação e exportação de dados.
O5. Analisar séries temporais e dados estatísticos utilizando dados reais da Pordata e INE.
O6. Resolver problemas práticos de análise de dados aplicando as bibliotecas numpy, pandas e matplotlib de forma integrada.

Programa

O programa está estruturado em cinco módulos sequenciais:
Módulo 1: Fundamentos de Python (→O1)
- Sintaxe básica, variáveis e tipos de dados
- Estruturas de controlo (condições, ciclos)
- Estruturas de dados (listas, dicionários, conjuntos)

Módulo 2: Manipulação de dados com pandas (→O2)
- Importação de dados de diferentes formatos (CSV, Excel, JSON)
- DataFrames e Series: criação, indexação e seleção
- Limpeza e transformação de dados
- Operações de agregação e agrupamento

Módulo 3: Visualização de dados (→O3)
- Introdução ao matplotlib
- Criação de gráficos básicos (linha, barras, histogramas, scatter plots)
- Personalização de gráficos e exportação

Módulo 4: Interface com Excel (→O4)
- Leitura e escrita de ficheiros Excel com pandas
- Manipulação de múltiplas folhas de cálculo
- Formatação e exportação de resultados

Módulo 5: Análise de dados reais (→O5, O6)
- Acesso e utilização de dados da Pordata e INE
- Análise de séries temporais
- Projeto integrador aplicando todas as competências desenvolvidas

Fórmula de cálculo da classificação final

A avaliação compreende as seguintes componentes:

1) Quizzes de progressão (QP): Ao longo da UC, os estudantes respondem a quizzes online para avançar entre módulos. Nota mínima para progressão: 70% em cada quiz.

2) Exame final (EF): Exame prático individual de 1h30m em sala de computadores, abrangendo todos os objetivos de aprendizagem (O1-O6).

Classificação final (CF) = 100% × EF

Nota: Os quizzes de progressão são obrigatórios para acesso ao exame final mas não contribuem para a classificação final. A classificação é expressa numa escala de 0 a 20 valores, sendo necessário obter pelo menos 10 valores para aprovação.

Obtenção de frequência

Para acesso ao exame final, os estudantes devem:
- Completar todos os quizzes de progressão com classificação mínima de 70%
- Participar nas sessões de apoio quando solicitado pelo docente (aplicável apenas a estudantes com dificuldades identificadas)
